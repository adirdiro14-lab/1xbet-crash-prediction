# 1xbet Crash Prediction

## Introduction
This project aims to predict crash events in the 1xbet gambling platform, focusing on the development of algorithms that can analyze betting patterns and predict the timing of crashes.

## Features
- Real-time data analysis
- Machine learning models for predictions
- User-friendly interface for interaction

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/adirdiro14-lab/1xbet-crash-prediction.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 1xbet-crash-prediction
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the prediction model:
```bash
python predict.py
```
